# Payment Analytics Dashboard

This is an **interactive Streamlit app** for visualizing and analyzing payment transaction data.  
It provides insights into:

- Payment method distribution (e.g., card, bank, wallet)
- Bank-wise payment counts
- Transaction success vs failure
- Failure reasons analysis

The app uses realistic sample data to simulate real-world payment scenarios.  
Charts and tables are created using **Plotly** for dynamic, interactive visualization.

## Features
- Clean and interactive dashboard interface
- Supports null/missing values gracefully
- Visualizes key metrics with bar charts, pie charts, and tables
- Easily extendable to real data sources
